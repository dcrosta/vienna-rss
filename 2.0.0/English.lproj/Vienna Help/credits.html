Credits

Vienna was written by Steve Palmer (steve@opencommunity.co.uk).

Additional code by Patrick Caulfield (tykepenguin@cix.co.uk).

Vienna is built using the following technologies:

  • Cocoa Framework using Objective-C.
  • SQLite public domain SQL database (http://www.sqlite.org).
  • MacPAD (http://macshareware.net/sdk.html) for the update technology.
  • Blackhole Media SQLDatabase 0.1 Objective-C wrapper for SQLite (http://blackholemedia.com/code/).

Special thanks are due to Jim Strong and Sam Aouane for their help in testing Vienna. Thanks are also due to Bryan Ford, Richard Buckle, Patrick Caulfield, Paul Wakeford, Nigel Martin, Jon Honeyball, D.J.Walker-Morgan, Ian Moran, Rod Mould, Rick Andrews, Adam Wood and Jane Palmer.
