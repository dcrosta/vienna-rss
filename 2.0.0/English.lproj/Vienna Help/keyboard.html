Keyboard Help

The following short cut keys are available:

Key Description
f Set focus to the search window.
r Mark the selected article read.
m Flag the selected article.
p Display the article's original web page.
< Display the previous viewed article.
> Display the next viewed article.
Spacebar Scrolls the article text if there is any more text to scroll, otherwise moves to the next unread article.
Enter Displays the next unread article.
Backspace Moves the selected articles to the trash folder. If this key is used in the trash folder, the selected articles are permanently deleted.
