Abbreviazioni da Tastiera

Puoi usare la tua tastiera per eseguire rapidamente diverse operazioni in Vienna. Per trovare le abbreviazioni per i comandi comuni, guarda nei menu (o vedi la lista in fondo a questa pagina). Molti elementi di Vienna come l'area cartella e l'elenco degli articoli hanno anche dei menu contestuali. Per vedere un menu contestuale, premi il tasto Control e fai clic sull'elemento.

Per eseguire un'azione, premi i tasti di abbreviazione sotto indicati.

Tasto Azione
Abbreviazioni a Tasto Singolo  
f Imposta il focus dell'input sulla finestra di ricerca.
r Segnala gli articoli selezionati come letti.
s Segnala la cartella corrente come letta poi passa alla cartella successiva con articoli non letti.
k Segnala la cartella corrente come letta.
m Segnala con contrassegno gli articoli selezionati.
< Mostra l'articolo o pagina web vista in precedenza.
> Mostra l'articolo o pagina web vista successivamente.
Spacebar Va all'articolo successivo non letto a meno che l'articolo corrente non abbia altro testo da scorrere ed in tal caso va su di una pagina all'interno dell'articolo corrente.
Comando-U Va all'articolo successivo non letto.
Enter/Return Apre l'articolo selezionato nella pagina web originale.
Delete/Backspace Sposta gli articoli selezionati nella cartella Cestino. Se questo tasto viene utilizzato nella cartella Cestino, gli articoli selezionati saranno eliminati in modo permanente.
Abbreviazioni Generali  
Comando-N Create a new subscription.
Shift-Comando-G Create a group folder.
Shift-Comando-F Create a smart folder.
Shift-Comando-E Edit the selected folder URL or edit a smart folder criteria.
Shift-Comando-D Delete the selected folder.
Shift-Comando-N Rename the selected folder.
Comando-R Refreshes all subscriptions.
Shift-Comando-R Refresh only those subscriptions selected in the folder list.
Comando-Meno Stops any active refresh.
Comando-P Print the selected articles.
Shift-Comando-P Brings up the Page Setup panel.
Shift-Comando-V Validate the selected feed.
Comando-? Displays the Vienna help book.
Abbreviazioni Articoli  
Shift-Comando-M Flags the selected articles.
Shift-Comando-U Marks the selected articles read.
Shift-Comando-S Marks the current folder read then skips to the next folder with unread articles.
Shift-Comando-O Restores an article from the trash back to its original folder.
Shift-Comando-K Marks all articles in the selected folders read.
Alt-Comando-K Marks all articles in all folders read.
Abbreviazioni Finestra  
Comando-W Closes the active tab window if one is open. Otherwise if no tabs are open, this closes the Vienna window.
Alt-Comando-W Closes all tab windows. Note that this command replaces the Close Tab command on the File menu when the Alt key is held down.
Control-Comando-Freccia Sinistra Displays the previous tab window.
Control-Comando-Freccia Destra Displays the next tab window.
Shift-Comando-W Closes the Vienna window but leaves Vienna running in the background. You can bring the window back by clicking on the Vienna dock icon or pressing Command-1.
Comando-M Minimizes the Vienna window to the dock.
Comando-0 Displays the activity log viewer window.
Comando-1 Reopens the Vienna window if it was previously closed.
Comando-2 Displays the Downloads window.
Abbreviazioni Finestra Browser  
Alt-R Reloads the current web page.
Comando-Punto Cancels loading of the current web page.
Comando-[/Comando-Freccia Sinistra Goes back to the previous web page.
Comando-]/Command-Freccia Destra Goes forward to the next web page.
Comando-F Puts the input focus in the search field. Entering some text here and pressing Enter will search for that text on the current web page.
Comando-G Searches for the next occurrence of the search text on the web page.
